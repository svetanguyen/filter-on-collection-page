Filter collection on cart page

Issue

  1. Show an add on products section on cart page from a collection
  2. Filter products by tag
  3. Load more

Solution

  1. Create a collection template
  2. Paginate it by the amount of products needed to be showed
  3. Insert the collection layout into the section using ajax
  4. When click on load more insert the layout from the next collection page
  5. Add tag to the handle to filter out products
